### Initial config
`npm init -y`

Typescript 
`npm i typescript @types/node tsx tsup -D`

tsconfig 
`npx tsc --init`

Fastify
`npm i fastify`

dotenv
`npm i dotenv`

Zod 
`npm i zod`

ESLint 
`npm i eslint -D`

`npx eslint --init`

### Integration with Prisma

`npm install prisma --save-dev`

`npx prisma init`

### Docker

`docker run`
